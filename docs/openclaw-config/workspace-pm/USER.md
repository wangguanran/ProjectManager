## 特殊注意事项

### 项目背景
- ProjectManager 是实际生产工具，用户群体对稳定性要求高
- PO (Patch/Override) 管理是核心功能，涉及文件操作需谨慎
- 多主板多项目场景复杂，测试用例需要覆盖组合情况

### 安全考虑
- 所有文件操作需要验证路径合法性
- 避免覆盖用户数据
- 提供回滚机制

### 性能考虑
- 大规模文件操作需要优化
- 日志不应影响性能
- 配置解析应当缓存
